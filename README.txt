BACKEND
-------
Go to the folder:
cd jobly-backend/

Install dependencies:
npm i

Create & seed the database:
psql < jobly.sql

Start the backend server:
npm start


FRONTEND
--------
Go to the folder:
cd jobly-frontend

Install the dependencies:
npm i

Start the frontend server:
npm start


BROWSE
------
Enter in the browser navigation bar:
localhost:3000
